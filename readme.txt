Logs plugin activities, shows status icons in Plugins page, and provides a full activity log page with filtering.

== Description ==
This plugin tracks plugin activations, deactivations, and updates, displays status icons with tooltips in the Plugins page, and provides a full log page with filtering by plugin, status, and date.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the activity log under Tools â†’ Plugin Activity Log.
